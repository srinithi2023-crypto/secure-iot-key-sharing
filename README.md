# Secure Secret Key Sharing for Resource-Constrained IoT Devices

## Overview

This project presents a lightweight security framework for resource-constrained IoT devices. It enables secure secret key sharing over MQTT while minimizing computational overhead and maintaining secure communication between connected devices.

---

## Features

- Lightweight secret key sharing
- MQTT-based communication
- Device authentication
- Secure message exchange
- Low computational overhead
